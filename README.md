# Customer Shopping Behavior Analysis

## Project Overview

This project analyzes customer shopping behavior to identify purchasing patterns, customer preferences, and business opportunities. The project demonstrates the complete data analytics process, from data cleaning to dashboard creation and business insights.

## Project Objectives

### Data Preparation & Exploratory Data Analysis (Python)

* Load and clean the dataset.
* Handle missing values and data inconsistencies.
* Explore customer behavior and purchasing trends using Python and Pandas.

### Data Analysis (SQL)

* Write SQL queries to analyze customer purchases.
* Identify customer segments and buying patterns.
* Generate insights on revenue, loyalty, and purchase behavior.

### Data Visualization (Power BI)

* Create an interactive dashboard to visualize key metrics and trends.
* Track customer demographics, sales performance, and purchasing habits.
* Present insights through clear and engaging visualizations.

## Tools Used

* Python (Pandas)
* SQL
* Power BI
* Jupyter Notebook
* GitHub

## Key Outcome

The project transforms raw customer data into meaningful insights that can help businesses understand customer behavior and make data-driven decisions.
